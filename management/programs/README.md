# Atlas Programs

Programs are the main execution units for Atlas.

Each program should contain:

- Mission
- Owner
- Priority
- Expected duration
- Scope
- Out of scope
- Architecture rules
- Deliverables
- Testing requirements
- Documentation requirements
- PR requirements
- Stop conditions
